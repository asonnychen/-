## 运行环境node v6.11.2
* 根目录的data.json是数据包
* 项目使用vue-cli手脚架构建
* css使用scss编译
* 大量注译可供参考
# 进入项目文件夹启动cmd
### * 安装前置包
> npm install
### * 运行开发版 服务器端口 localhost:8888
> npm run dev
### * 打包生产运行
> npm run build
